# Skill-based-course-allocation-sysyem-java-code-
This project is a **Skill-Based Course Allocation System** built using Java with a strong focus on Object-Oriented Programming concepts. The main idea behind the system is pretty simple: students have certain skills and marks, courses have requirements, and the system matches them in a smart way. So instead of randomly assigning courses, it actually checks whether a student has the required skill and meets the minimum marks before allocating anything. This makes the whole process more logical and realistic.

The application is designed as a **menu-driven console system**, which makes it easy to use and test. You can add students, add courses, display all student details, allocate courses based on eligibility, update student information, and even save/load data using file handling. The allocation logic is the core part of the system — it loops through each student and each course, checks conditions like skill match and marks, and then assigns the appropriate courses. If no course matches, it clearly shows that as well, so the output is always meaningful.

From a design perspective, the project properly uses **OOP principles**. There is a clear separation of responsibilities: data classes like `Student` and `Course` handle the data, the allocation logic is handled separately (through classes like `AllocationSystem`), and file operations are managed using a dedicated `FileManager`. It also demonstrates concepts like **inheritance and abstraction** (for example, using abstract classes like `Person` and `Allocation`), making the design more modular and easier to expand in the future.

To make the system easier to understand and present, multiple **UML diagrams** have been included — such as class diagrams (to show structure), use case diagrams (to show user interaction), activity diagrams (to show flow of control), and sequence diagrams (to show how components interact step-by-step). These diagrams help visualize how everything connects together behind the scenes.

Overall, this project is a clean implementation of a real-world scenario using Java. It focuses on clarity, proper structure, and applying core programming concepts rather than just making something that works. It can also be extended further in the future by adding features like GUI, database integration, or multiple user roles.
